# Sistema-AT-Univap
Sistema desenvolvido em Python para automatizar a leitura das planilhas de notas técnicas (AT) dos alunos dos cursos técnicos do Colégio Técnico Univap. O programa lê um arquivo Excel, calcula as médias bimestrais, gera relatórios personalizados em HTML e PDF, e destaca visualmente o desempenho de cada aluno.
